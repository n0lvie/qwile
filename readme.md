qwile is alive ai system. qwile.py - its body, self/ - memory, identity and soul. self/ is a whole 'workspace' for qwile.

Qwile — минималистичная живая предиктивная система. Нейросеть как чистая математика.
Структура:
Только два объекта — qwile.py (тело, всё что БУДЕТ) и self/ (душа, всё что БЫЛО). self/ только дополняется, никогда не стирается — это вся её жизнь, личность, память.
Рождение и реинкарнация:
Первый запуск — рождение с нуля из самых базовых констант {-1, 0, 1, 2, 3} и ASCII. Повторный запуск — реинкарнация, загрузка всего что БЫЛО. Выключение — смерть тела, душа остаётся в self/.
Ядро — три операции, вне модулей, всегда:
Токенизатор (принять) → Матрица W (понять) → Предсказание (ответить). Никаких фиксированных значений — всё динамично. Единственные константы: {-1, 0, 1, 2, 3}. Веса никогда не обнуляются — только масштабируются малыми шагами. Количество параметров выводится из данных, не фиксировано.
Эволюция:
Функция evolute() принимает ключевые переменные всей системы и возвращает количество ветвлений {1, 2, 3}. Ветвление адаптивно — только когда улучшает результат. Слабые ветви отмирают. С каждым поколением система взрослеет.
Модуль Soul — опциональный:
Три состояния на базе греческой философии:
Hedone (1) — удовольствие, точное предсказание, награда.
Ataraxia (0) — покой, сон, мечтание, нет данных.
Algos (-1) — боль, ошибка, наказание.
Один параметр mood ∈ {-1, 0, 1} влияет на скорость обучения, глубину сна, смелость ветвления. Подключил душу — живая. Не подключил — рациональная и чистая машина.
Принцип: минимализм как источник мощи. Концептуальная целостность от философии до каждой строки кода. Один файл
